# TUI 使用指南

## 启动

```bash
dumbcoder tui
```

需要在包含 `.dumbcoder/` 或 `.git/` 的项目目录中运行。

## 界面布局

```
┌─────────────────────────────────────────────────────┐
│ dumbcoder │ Ready │ 2 messages │ Ctrl+C: quit...    │
├──────────────────┼──────────────────────────────────┤
│                  │                                  │
│  Chat            │  Context                         │
│                  │                                  │
│  > 你的问题       │  ── Symbols ──                   │
│  < AI 回答        │  function cancel_order           │
│                  │    src/order/service.rs:42       │
│                  │                                  │
│                  │  ── Referenced Files ──          │
│                  │  ── src/order/service.rs ──      │
│                  │  pub fn cancel_order(...)        │
├──────────────────┴──────────────────────────────────┤
│  > 输入问题...                           [Enter] Send│
└─────────────────────────────────────────────────────┘
```

## 使用方法

1. 在底部输入栏中输入问题
2. 按 `Enter` 发送
3. AI 回答会显示在左侧面板
4. 引用的文件和符号会显示在右侧面板

## 快捷键

| 快捷键 | 功能 |
|--------|------|
| `Enter` | 发送消息 |
| `Tab` | 切换焦点面板（聊天 / 上下文） |
| `Esc` | 切换面板，若输入为空则退出 |
| `Up` / `Down` | 在上下文面板中滚动 |
| `PgUp` / `PgDn` | 上下文面板翻页 |
| `Ctrl+C` | 退出 TUI |
| `Ctrl+L` | 清空对话历史 |
| `Backspace` | 删除光标前字符 |
| `Delete` | 删除光标后字符 |
| `Left` / `Right` | 移动光标 |
| `Home` / `End` | 跳到行首/行尾 |

## 状态指示

- **Ready** — 绿色，可以输入新问题
- **Thinking...** — 黄色 + 旋转指示器，正在等待模型响应
- **Error** — 红色，发生错误（会在聊天面板显示错误信息）

## 使用场景

- **代码问答**: 询问项目结构、函数职责、调用关系
- **代码解释**: 通过 `explain` 命令查看特定文件或函数的解释
- **代码搜索**: 通过 `search` 命令搜索相关代码
- **多轮对话**: 支持连续提问，AI 会参考上下文

## 前置条件

1. 需要先运行 `dumbcoder init` 初始化项目
2. 需要本地 Ollama 服务运行（默认 `http://127.0.0.1:11434`）
3. 建议先运行 `dumbcoder index` 建立代码索引以获得更好的结果
