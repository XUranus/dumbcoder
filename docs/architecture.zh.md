# dumbcoder 系统架构

## 总体架构

dumbcoder 采用分层架构设计，从上到下分为五层：

```
┌──────────────────────────────────────────────┐
│              Rust CLI / TUI                  │
│  ask / explain / search / test / review / patch│
└──────────────────────────────────────────────┘
                     │
                     v
┌──────────────────────────────────────────────┐
│           Agent Orchestrator                 │
│  任务拆分 / 上下文管理 / 工具调用 / 安全策略   │
└──────────────────────────────────────────────┘
                     │
                     v
┌──────────────────────────────────────────────┐
│              Tool Layer                      │
│ rg / git / tree-sitter / LSP / test / diff    │
└──────────────────────────────────────────────┘
                     │
                     v
┌──────────────────────────────────────────────┐
│           Code Index & RAG Layer             │
│ BM25 / Vector DB / AST Index / Git History    │
└──────────────────────────────────────────────┘
                     │
                     v
┌──────────────────────────────────────────────┐
│           Local Model Service                │
│ Ollama / llama.cpp / vLLM / SGLang            │
│ Qwen2.5-Coder-7B / 14B / other local models   │
└──────────────────────────────────────────────┘
                     │
                     v
┌──────────────────────────────────────────────┐
│           Security & Audit Layer             │
│ 文件权限 / 命令白名单 / patch 审计 / 日志       │
└──────────────────────────────────────────────┘
```

## 核心模块

| 模块 | 文件 | 职责 |
|------|------|------|
| CLI 入口 | `main.rs` | 命令行解析、子命令调度 |
| 配置管理 | `config.rs` | 加载/保存 TOML 配置 |
| 模型客户端 | `model.rs` | 调用 Ollama API |
| 代码上下文 | `context.rs` | 组装代码上下文、格式化提示词 |
| 代码搜索 | `cmd/search.rs` | 调用 ripgrep 搜索 |
| 代码问答 | `cmd/ask.rs` | 搜索 + 上下文 + 模型问答 |
| 代码解释 | `cmd/explain.rs` | 文件/函数解释 |
| 安全过滤 | `security.rs` | 文件黑名单、路径沙箱、命令白名单 |
| 工具函数 | `util.rs` | 终端输出格式化、项目检测 |

## 数据流

用户输入问题 → ripgrep 搜索相关代码 → 安全过滤 → 上下文组装 → 调用本地模型 → 输出格式化回答

## 技术栈

- **语言**: Rust
- **CLI 框架**: clap (derive)
- **异步运行时**: tokio
- **HTTP 客户端**: reqwest
- **配置**: serde + toml
- **模型服务**: Ollama (本地)
- **代码搜索**: ripgrep
